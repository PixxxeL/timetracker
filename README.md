# Timetracker

You must have Node.js and NPM

For developement:

* Change directory to repository root
* Run `npm install -g bower gulp`
* Run `bower install`
* Run `npm install`
* Run `gulp`
* In browser type `http://127.0.0.1:8090/html/`

For using:

* Change directory to repository root
* Run `npm install -g bower gulp`
* Run `bower install`
* Run `npm install`
* Run `gulp build`
* Run `gulp demo`
* In browser type `http://127.0.0.1:8090/`
